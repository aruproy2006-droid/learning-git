# Git Basics

## What is Git?

Git is a version control system that helps developers track changes in their code and collaborate with others.

---

## What is GitHub?

GitHub is a cloud-based platform where developers store and manage Git repositories.

---

## Common Git Commands

### Initialize a Repository
```bash
git init
```
Creates a new Git repository.
### Clone a Repository
```bash
git clone <repository-url>
```
Downloads a repository from GitHub to your computer.
### Check Status
```bash
git status
```
Shows the current state of your repository.
### Add Files
```bash
git add .
```
Adds all changed files to the staging area.
### Commit Changes
```bash
git commit -m "Initial commit"
```
Saves changes with a message.
### Push Changes
```bash
git push
```
Uploads commits to GitHub.
### Pull Changes
```bash
git pull
```
Downloads the latest changes from GitHub.
