# Git Basics

## What is Git?

Git is a version control system that helps developers track changes in their code and collaborate with others.

---

## What is GitHub?

GitHub is a cloud-based platform where developers store and manage Git repositories.

---

## Common Git Commands

### Initialize a Repository

```bash
git init
```

### Clone a Repository

```bash
git clone <repository-url>
```


### Check Status

```bash
git status
```
